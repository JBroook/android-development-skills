# Super Silent

<img width="100" height="100" alt="Logo_upscale" src="https://github.com/user-attachments/assets/6e330fff-836e-4bb2-acc3-36845807955d" />

A silent SOS Android app developed in Java for a university course by a single second year Computer Science student

## Features
- Firebase registration and login
- Volume key press sequence detection even outside the app
- Set custom key press sequence
- Automatically sends custom emergency SMS to chosen contacts
- Includes current location in emergency SMS

## Requirements
- Android Studio Narwhal
- Android device with sim card or emulator

## Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/jbroook/android-development-skills
   ```
2. Sync Gradle file in Android Studio
3. Run the app
